🔍 Amalgam Debug Suite

Manual diagnostic tools for step-by-step problem isolation

🎯 Purpose of Debug Suite

Manual Debugging provides interactive tools for diagnosing specific issues during development. Debug tools allow real-time exploration and provide detailed logging for LLM assistance.

  • Interactive Exploration: Manual control over diagnostic processes
  • Visual Inspection: Canvas rendering and state visualization
  • Real-time Problem Isolation: Step-by-step issue diagnosis
  • LLM-Friendly Output: Copy-paste functionality for error sharing

📊 Data Inspector

Analyze board data structure, verify JSON loading, and check data integrity. Perfect for diagnosing data corruption issues.

Use when: Data loading fails, intersections missing, board structure issues

Inspect Data Structure

🎨 Board Analysis

Step-by-step board rendering diagnosis with canvas inspection, pixel sampling, and intersection verification.

Use when: Board not rendering, missing lines/intersections, canvas issues

Analyze Board Rendering

⚡ Comparison Tool

Side-by-side comparison between main page and debug rendering to identify discrepancies.

Use when: Main page differs from expected, inconsistent rendering
